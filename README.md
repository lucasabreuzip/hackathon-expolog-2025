Plataforma de empregabilidade do Porto do Pecém - Conectando profissionais qualificados com certificações técnicas às melhores oportunidades no Complexo Industrial e Portuário.

A **Pecém +Empregabilidade** é uma plataforma web moderna desenvolvida para conectar talentos qualificados com empresas do Complexo Industrial e Portuário do Pecém, no Ceará.
